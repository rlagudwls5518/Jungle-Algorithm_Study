1. 로그인 / 회원가입 페이지
페이지 상단에 “로그인 / 회원가입” 제목이 크게 표시됨.

로그인 섹션

아이디 / 비밀번호 입력 필드

아래에 로그인, 회원가입 버튼

JWT 방식으로 로그인 처리

아이디 또는 비밀번호가 하나라도 틀리면 오류 메시지 표시

2. 회원가입 – 내 정보 입력 페이지
왼쪽 상단: 프로필 사진 업로드 박스

오른쪽:

별명 입력

나이 입력

성별 선택

하단:

아이디 / 비밀번호 입력

비밀번호 아래: 한줄소개 입력란

최하단: 저장 버튼

저장 시 DB에 별명, 나이, 성별, 한줄소개, 아이디/비밀번호가 함께 저장됨 (JWT 계정 생성 포함)

3. 밸런스 게임 메인 게시판
상단 박스 (헤더+질문입력)

서비스명 + 글 입력칸

질문 입력 (예: "짜장 vs 짬뽕")

작성하기 버튼
→ 하나의 큰 입력 박스로 묶여 있음

그 아래

가장 좋아요를 많이 받은 질문 1개가 하이라이트로 노출됨

그 아래

최신 등록된 질문들이 리스트 형태로 정렬 (최신순 기본 정렬)

4. 질문 카드 UI 구성
각 질문은 하나의 카드 형태로 구성됨:

질문 박스1 vs 질문 박스2
→ 아래에 각각 투표 선택지 버튼 2개

투표 후
→ 각 선택지 박스가 투명도 있게 변하고, 투표 수 표시됨 (몇 명이 선택했는지)
