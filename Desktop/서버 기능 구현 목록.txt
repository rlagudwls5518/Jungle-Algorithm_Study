서버 기능 구현 목록


1. 로그인할때 JWT방식으로 로그인
2. 회원가입버튼클릭시  회원가입 페이지로 이동 
3. 로그인버튼을 누르면 아이디 비밀번호 확인후 게시판화면으로 이동

4. 회원가입페이지에서 아이디, 비밀번호,별명, 나이, 한줄 소개를 적을수 있고 저장버튼을 누르면 데이터 베이스에 저장후 게시판화면으로 이동

5. 게시판 화면에서 밸런스게임 카드 빈칸두개의 내용을 입력해서 데이터베이스에 저장
6. 게임카드는 최신작성순으로 볼수있도록
7. 게임카드에 좋아요를 누르면 서버에서 좋아요수 증가
8. 게임카드 수정을 누르면 두개의 빈칸을 다시 만들어 수정할 글 입력
9. 게임카드에서 1번 버튼 2번버튼 누르면 +1되고  몇명이 투표를 했는지 서버에서 가져와서 보여줌
